# WonderLust - Explore and Share New Places  

WonderLust is a dynamic website where users can discover, add, and review listings for various places, including hotels, restaurants, and more. It empowers users to contribute to the community by sharing insights and reviews, helping others make informed decisions about the listed places.  

---  

## 🚀 Features  
- **Add Listings**: Create new listings for places, hotels, restaurants, etc.  
- **Manage Listings**: Edit or update existing listings with ease.  
- **Reviews**: Add reviews for various places to help others decide.  
- **Search Functionality**: Quickly search for specific listings.  
- **User Authentication**: Sign up, log in, and log out to manage personalized experiences.  
- **Session Management**: User sessions are stored securely in MongoDB using `connect-mongo`.  

---  

## 🛠️ Technologies Used  
WonderLust is built using the following technologies:  
- **Frontend**:  
  - HTML  
  - CSS  
  - Bootstrap  
  - JavaScript  

- **Backend**:  
  - Express.js  
  - Node.js  
  - Passport.js for user authentication  

- **Database**:  
  - MongoDB (Hosted on MongoDB Atlas)  

- **Session Management**:  
  - `connect-mongo` for storing session information in MongoDB  

- **Image Hosting**:  
  - Cloudinary  

- **Deployment**:  
  - Render

## 📖 How to Use WonderLust  
1. **Home**: Browse the homepage for an overview of available listings.  
2. **All Listings**: View all the places added by the community.  
3. **Add Listing**: Create a new listing with details about the place.  
4. **Search**: Use the search feature to quickly find specific listings.  
5. **Sign Up**: Create an account to add or review listings.  
6. **Login**: Log in to your account to access personalized features.  
7. **Logout**: Securely log out from your account when done.

## 🧑‍💻 Authentication and Session Management  
- **Authentication**:  
  WonderLust uses `Passport.js` for user authentication. It allows users to securely sign up and log in to the website.  
- **Session Management**:  
  User session data is stored in MongoDB using `connect-mongo`, ensuring secure and persistent session storage.
